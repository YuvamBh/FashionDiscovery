Browse → Want → Buy → Satisfied → Leave
(Value = transaction)

Discover → Signal → Influence → See Impact → Return
(Value = cultural participation)

### Why Brands Pay Without GMV

Traditional marketing:
Spend → Hope → Launch → Measure Sales → Regret

This compounds WITHOUT needing transaction volume.

## CRITICAL QUESTIONS ANSWERED

**Q: Won't users get bored without buying?**
A: 
- Instagram has no checkout (still engaging)
- Pinterest has delayed purchase (still valuable)
- Our influence feedback is MORE satisfying than buying
- Moodboards become identity, not shopping lists

**Q: Won't brands just use this then advertise elsewhere?**
A:
- The signals are unique (can't get elsewhere)
- The launch confidence is addictive
- Avoiding dead stock = direct ROI
- Historical data becomes moat

**Q: What if brands don't launch what tested well?**
A:
- We track this
- Show brands who ignore signals fail more
- Builds reputation system
- Eventually, API access requires proof of action

**Q: How do you prevent fake engagement?**
A:
- Authority scoring (consistent taste = higher weight)
- Pattern detection (random swiping = low weight)
- Segment coherence checks
- Time-based analysis
- Manual curation at scale
