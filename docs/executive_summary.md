## CRITICAL SUCCESS FACTORS (RANK ORDER)

```
1. SIGNAL QUALITY (Most Critical)
   Without good signals, nothing else matters
   → Obsess over authority scoring
   → Fraud detection from day 1
   → Taste consistency validation

2. USER RETENTION (Critical)
   No purchase hook means retention is HARD
   → Influence feedback must be compelling
   → Weekly fresh content non-negotiable
   → Social validation essential

3. BRAND REPEATABILITY (Critical)
   One-time experiments don't build business
   → First experiment must show clear ROI
   → Support brands like consultants
   → Make insights actionable

4. SPEED TO MARKET (Very Important)
   First-mover advantage significant
   → MVP in 8 weeks max
   → Iterate fast
   → Beat copycats with network effects

5. TEAM QUALITY (Very Important)
   Small team needs to be excellent
   → Technical excellence (feed, signals)
   → Design excellence (feel matters)
   → Sales capability (consultative)

6. CAPITAL EFFICIENCY (Important)
   Can't outspend way to success
   → Organic growth focus
   → High gross margins
   → Prove model before scaling spend
```

## THE UNFAIR ADVANTAGES

```
1. TIMING
   → D2C brands proliferating (supply side ready)
   → Gen Z wants discovery, not deals (demand side ready)
   → Social commerce hasn't solved this yet (gap exists)

2. APPROACH
   → Neutral platform (not selling, not biased)
   → Pre-purchase intent (earlier in funnel)
   → Non-transactional engagement (unique loop)

3. NETWORK EFFECTS
   → More users = better signals = happier brands
   → Happier brands = better products = happier users
   → Compounds over time

4. DATA MOAT
   → Accumulates daily
   → Proprietary & predictive
   → Hard to replicate
   → Increases in value

5. CATEGORY CREATION
   → Not competing in existing category
   → Creating "cultural demand intelligence"
   → Define the rules
```

## THE BOTTOM LINE

This is a **high-conviction, high-execution** idea.

**It works if:**
- Signals genuinely predict launches (provable in Month 3)
- Users return without buying (validated by retention metrics)
- Brands see ROI and repeat (proven by repeat rate)

**It fails if:**
- Signals are noise (low intent rate, random patterns)
- Users churn fast (retention < 25%)
- Brands don't repeat (one-time value only)

**The unfair advantage:**
- Network effects that compound
- Data moat that grows daily
- Category creation (not competition)

**The risk:**
- Retention without transaction is HARD
- Requires excellent execution
- No room for mediocrity

**The opportunity:**
- Large market (₹500Cr+ D2C fashion in India)
- Clear wedge (fashion → lifestyle → electronics)
- Defensible moat (data + network)
- Timing is right (D2C boom + Gen Z discovery)

**Next step:**
Build the MVP. Test with 500 users. Prove signals work.
Everything else is noise until that's validated.
